# Stellog 用户手册

Stellog 是一款本地习惯打卡 App，帮助你创建习惯、每天打卡、查看日历记录、解锁成就，并提供智能推荐与 AI 助手。所有数据保存在本机，无需登录、无需联网（AI 助手除外）。

## 1. 启动

打开应用后会先显示启动页（Logo 与应用名），随后自动进入主页面。

## 2. 底部导航

底部共有五个入口：

- **首页**：活动卡片 / 列表、打卡。
- **日历**：按月查看打卡记录、补打卡、按活动筛选。
- **AI助手**：与 AI 对话（需先配置，见第 8 节）。
- **成就**：查看已解锁 / 未解锁的勋章。
- **我的**：账户数据、AI 设置、智能推荐开关、深色模式。

## 3. 创建 / 编辑 / 删除活动

- **创建**：点击首页右下角的「＋」，输入活动名称（必填）和计量单位（可选，如 分钟、km、个），点击保存。
- **编辑**：长按活动卡片（或列表项）→ 选择「编辑活动」，修改后保存。
- **删除**：长按 → 选择「删除活动」→ 确认。该活动的打卡记录会一并删除。

## 4. 打卡与记录

- 在卡片上点击「打卡」即可完成今日打卡，连续天数、本周圆点、累计值会同步更新。
- 打卡后可点击「记录详细」填写今天完成的数量（按你设置的单位）。
- 点击「取消」可撤销今日打卡（有二次确认）。
- 卡片描边与当天圆点会提示今日是否已完成（绿色＝已完成，红色＝待完成）。

## 5. 卡片 / 列表切换与浏览

- 顶部「卡片 / 列表」可切换两种展示方式。
- 卡片模式下左右滑动切换不同活动，下方圆点指示当前位置。

## 6. 日历与补打卡

- 在首页点击卡片上的「本周打卡情况」区域，或点击底部「日历」进入日历页。
- 点击某一天查看当天各活动的完成情况。
- 点击过去的日期，可对活动进行**补打卡**（有二次确认），也可填写数量。
- 点击顶部「全部活动」按钮可筛选要在日历中显示 / 统计的活动。

## 7. 提醒

- 点击卡片顶部的闹铃图标进入提醒编辑页。
- 设置提醒标题（默认带入活动名称）、开始与结束时间，保存即可。已设提醒的卡片闹铃图标会高亮。

## 8. AI 助手与设置

- 点击底部「AI助手」进入对话页，可询问习惯养成、活动安排、打卡复盘等问题，回答会结合你本周的打卡数据。
- 首次使用需在「我的 → AI设置」中配置：
  - **API 地址**：默认 `https://api.deepseek.com/chat/completions`，也可填任意 OpenAI 兼容地址（含本地模型）。
  - **模型**：默认 `deepseek-chat`。
  - **API Key**：使用默认 DeepSeek 地址时必填；改为本地等自定义地址时可留空。
- 使用 AI 助手需要联网。

## 9. 智能推荐排序

- 在「我的」页打开「智能推荐排序」后，首页会根据你近期的打卡情况（中断风险、完成频率、连续天数）调整活动顺序，并在卡片上给出简短提示。
- 关闭后按创建顺序展示。

## 10. 成就

- 随着累计打卡次数、连续天数、活动数量等达到条件，会自动解锁对应勋章，可在「成就」页查看解锁时间与条件。

## 11. 深色模式

- 在「我的 → 深色模式」中可选择 浅色 / 深色 / 跟随系统，默认浅色。

## 12. 账户数据

- 「我的」页顶部显示创建活动数、累计打卡次数、历史最长连续天数。

## 常见问题

- **AI 助手提示请求失败**：请检查网络、API 地址、模型名称、API Key 或账户余额。
- **AI 助手提示请先填写 API Key**：使用默认 DeepSeek 地址时需要先在 AI 设置中填写 Key。
- **提醒没有弹出通知**：当前版本提醒仅用于记录设置，暂不触发系统通知。
- **数据是否会丢失**：活动、打卡、补打卡、成就等都保存在本机数据库，重启后仍在。
